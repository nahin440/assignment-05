Important FAQ ?

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming API for HTML and XML documents.

How do you select an element from the DOM?

There are several ways to select an element from the DOM such as :

  • 1. By ID
  • 2. By Class Name
  • 3. By Tag Name
  • 4. By CSS selector (querySelector)
What is event delegation in the context of the DOM,and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an element's attributes and styles using the DOM?

The classList object only provide methods to change the class of an element. If you want to change other attributes like id , href , or src , you can use the setAttribute() method.
